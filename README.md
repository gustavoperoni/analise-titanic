# 🚢 Análise de Dados do Titanic

Projeto de análise exploratória de dados utilizando Python e Pandas.

---

## 🎯 Objetivo
O objetivo deste projeto é praticar análise de dados, explorando, limpando e analisando um conjunto de dados real do Titanic.

---

## 🧠 Ferramentas utilizadas
- Python  
- Pandas  
- Jupyter Notebook  

---

## 📊 Análises realizadas
- Sobrevivência por gênero  
- Sobrevivência por classe social  
- Sobrevivência por faixa etária  

---

## 🧹 Etapas do projeto
- Limpeza e tratamento de dados ausentes  
- Remoção de colunas desnecessárias  
- Criação de faixas de idade  
- Análise exploratória dos dados  

---

## 📌 Conclusão
Foi possível identificar que fatores como gênero, classe social e idade tiveram grande influência na sobrevivência dos passageiros do Titanic.
